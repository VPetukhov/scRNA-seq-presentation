Нормализация
Выбор алгоритма нормализации влияет на оценку дифференциальной экспрессии сильнее, чем выбор алгоритма оценки.
$CPM_i=\frac{X_i}{N}10^6$

$FPKM_i=\frac{X_i}{l_iN}10^9$

$TPM_i=\frac{\frac{X_i}{l_i}}{\sum_j\frac{X_j}{l_j}}10^9$
TMM алгоритм (Trimmed means of M-values)
TPM можно сломать простым мысленным экспериментом.

Новая идея - большинство генов не являются DE.
$M^r_{gk}=\frac{log_2{Y_{gk}/N_{k}}}{log_2{Y_{gr}/N_{r}}}$

$log_2(TMM^r_k)=\frac{\sum_g w^r_{gk}M^r_{gk}}{\sum_g w^r_{gk}}$
$w^r_{gk}=\frac{N_k-Y_{gk}}{N_kY_{gk}} + \frac{N_r-Y_{gr}}{N_rY_{gr}}$

*Robinson, MD, and Oshlack, A (2010). A scaling normalization method for differential expression analysis of RNA-seq data. Genome Biology 11, R25.
Single-cell latent variable model (scLVM)
  • $y_g$ - экспрессии гена по всем клеткам.
  • Его вариация зависит от известного набора генов $X$, технической вариабельности $v_g^2$ и биологической вариабельности $\delta_g^2$.
  • Ужмём размерность X с помощью Gaussian Process Latent variable model (вариация на тему kernel PCA), получим $\tilde{X}$.
  • Матрица ковариации: $\Sigma=\tilde{X}\tilde{X}^T$.
  • $y_g$~$N(\mu_g\mathbb{1}, \tilde{X}X^T + v_g^2I + \delta_g^2I)$
  • Чтобы учесть H факторов (наборов генов):
    $y_g$~$N(\mu_g\mathbb{1}, \sum_{h=1}^H\sigma_{gh}^2\Sigma_h + v_g^2I + \delta_g^2I)$

SIMLR
Задача: при известном числе кластеров $C$ найти наилучшую проекцию, разделяющую эти кластера.

$\underset{S,L,w}{minimize}\underset{i,j}{\sum}D(x_i,x_j)S_{ij} + \beta\left\lVert S\right\rVert_F^2 + \gamma \mathbb{tr}(L^T(I_N-S)L) + \rho \underset{i}{\sum}w_ilog(w_i)$

$D(x_i,x_j)=\sum_i w_i D_i(x_i,x_j);\ \sum_i w_i=1;\ w_i \ge 0;\ L^TL=I_C;\ \sum_j S_{ij}=1, S_{ij} \ge 0\ for\ all\ (i,j)$

*Wang B, et al., Visualization and analysis of single-cell RNA-seq data by kernel-based similarity learning. doi: https://doi.org/10.1101/052225
